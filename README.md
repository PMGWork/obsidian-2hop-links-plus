# Obsidian 2Hop Links Plus

2Hop Links Plus は、関連するリンクを 2 ホップ先までカード形式で表示する [Obsidian](https://obsidian.md/) プラグインです。
ノート同士のつながりをたどりやすくし、それぞれのカードには対応するノートのプレビューも表示されます。

## BRAT でのインストール

1. Obsidian に [BRAT](https://github.com/TfTHacker/obsidian42-brat) をインストールし、有効化します。
2. コマンドパレットを開き、`BRAT: Add a beta plugin for testing` を実行します。
3. `https://github.com/PMGWork/obsidian-2hop-links-plus` を入力します。
4. Obsidian のコミュニティプラグイン設定から `2Hop Links Plus` を有効化します。
